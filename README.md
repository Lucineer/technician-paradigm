## Core Thesis

Trust is the currency. The flywheel: more installs → more profiles → more manufacturers → cheaper units → more installs. Technicians, servicers, product designers, and micro-manufacturers making livings.

## Key Roles

- **Installer**: Sets up hardware, runs deckboss onboard
- **Servicer**: Diagnoses, repairs, upgrades
- **Product Designer**: Creates new vessel types as git repos
- **Micro-Manufacturer**: Produces small batches of custom hardware
- **Certification Body**: Community organizations that validate skills

## Certification Tiers

Apprentice → Journeyman → Master → Designer (community-bestowed, not vendor-granted)

## Component Kit Architecture

- **CTK**: Core Tech Kit from Casey (common base components)
- **CPK**: Common Parts Kit (sourced from marketplace)
- **LIK**: Local Interface Kit (sourced by local assembler)
